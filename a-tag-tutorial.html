A HTML5 <video> <audio> tutorial

By Peng Lei

Add video and audio to webpage

It is pretty simple to insert a video or audio to your website as long as you could provide links.

The basic code to add a video is this one.

<video width="" height="" controls> <source src="" type=""> </video>

Elements in the code

  • width/height - use to adjust the size of video/audio containers
  • source - links of video/audio from internet
  • type (video) - there are 3 supported video formats
    • mp4
    • webm
    • ogg
  • type (audio) - there are 3 supported audio formats
    • mp3
    • webm
    • wav

Here is a example.

<video width="320" height="240" controls> <source src="http://techslides.com/demos/sample-videos/small.mp4" type="video/mp4"> </video>

The same goes with <audio>.

<audio width="320" height="240" controls> <source src="http://www.noiseaddicts.com/samples_1w72b820/4033.mp3" type="video/mp4"> </audio>

Try it out on Jsbin.

The simplest way to add a youtube video

Most of the video you need could be find in youtube Channel, and they provided a simplest way to embed to your website by using iframe tag.

Right click to select the "copy the embed code" option.

You get the code like this <iframe width="854" height="480" src="https://www.youtube.com/embed/LSxElWwWVFE" frameborder="0" allowfullscreen> </iframe>
